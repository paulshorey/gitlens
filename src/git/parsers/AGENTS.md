# git/parsers

Parse raw `git` output into models.

- **parsers.ts** – Parser exports
- **logParser.ts** – `git log` output
- **blameParser.ts** – `git blame` output
- **branchParser.ts** – `git branch` output
- **remoteParser.ts** – `git remote` / URL parsing
- **tagParser.ts** – `git tag` output
- **diffParser.ts** – Diff output
- **reflogParser.ts** – Reflog
- **statusParser.ts** – Status (ahead/behind)
- **stashParser.ts** – Stash list
- **shortlogParser.ts** – Shortlog
- **treeParser.ts** – Tree
