- **logCommit.ts** – Log commit

- **repository.ts** – Repository (branches, remotes, status)
- **blame.ts** – Blame result
- **log.ts** – Log result
- **diff.ts** – Diff
- **file.ts** – File in commit
- **tree.ts** – Tree
- **author.ts** – Author
- **contributor.ts** – Contributor
- **remoteProvider.ts** – Remote provider base
- **pullRequest.ts** – Pull request
- **issue.ts** – Issue
- **reflog.ts** – Reflog
- **rebase.ts** – Rebase state
- **merge.ts** – Merge
- **stash.ts** – Stash
- **shortlog.ts** – Shortlog
- **defaultBranch.ts** – Default branch
- **user.ts** – User
- **status.ts** – Status
