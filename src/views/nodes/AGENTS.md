# views/nodes

Tree node classes for sidebar views. Each node extends ViewNode.

- **viewNode.ts** – Base ViewNode
- **common.ts** – Shared node helpers
- **helpers.ts** – Node helpers
- **repositoryNode.ts** – Repository; **repositoriesNode.ts** – Repo list
- **commitNode.ts** – Commit; **stashNode.ts** – Stash
- **branchNode.ts** – Branch; **branchesNode.ts** – Branch list
- **remoteNode.ts** – Remote; **remotesNode.ts** – Remote list
- **tagNode.ts** – Tag; **tagsNode.ts** – Tag list
- **contributorNode.ts** – Contributor
- **fileHistoryNode.ts** – File history entry
- **lineHistoryNode.ts** – Line history
- **pullRequestNode.ts** – Pull request
- **statusFileNode.ts** – Status file
- **compareBranchNode.ts** – Compare branch
- **mergeConflict*.ts** – Merge conflict nodes
- **rebaseStatusNode.ts** – Rebase status
- **reflogNode.ts** – Reflog
- **searchResultsNode.ts** – Search results
- **folderNode.ts** – Folder
