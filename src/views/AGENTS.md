# views

Sidebar tree views. Base: `viewBase.ts`; commands: `viewCommands.ts`.

- **commitsView.ts** – Commits view
- **repositoriesView.ts** – Repositories view
- **branchesView.ts** – Branches view
- **remotesView.ts** – Remotes view
- **stashesView.ts** – Stashes view
- **tagsView.ts** – Tags view
- **contributorsView.ts** – Contributors view
- **fileHistoryView.ts** – File history view
- **lineHistoryView.ts** – Line history view
- **searchAndCompareView.ts** – Search & compare
- **nodes.ts** – Node type exports
- **viewBase.ts** – Base TreeDataProvider
- **viewCommands.ts** – View command handlers
- **viewDecorationProvider.ts** – Decorative icons for nodes
- **nodes/** – Tree node implementations
- **commits-panel/** – Kylin: CommitsPanel, FileHistoryChartViewPanel (Chart.js webviews)
