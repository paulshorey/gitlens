# system

Shared utilities and helpers.

- **string.ts** – interpolate, escapeMarkdown, base64, md5, sha1, truncate, etc.
- **array.ts** – Array utilities
- **object.ts** – Object utilities
- **date.ts** – Date formatting
- **path.ts** – Path utilities
- **version.ts** – Version comparison
- **encoding.ts** – URL encoding
- **function.ts** – Function utilities
- **iterable.ts** – Iterable helpers
- **promise.ts** – Promise utilities
- **searchTree.ts** – Tree search
- **stopwatch.ts** – Timing
- **decorators/** – @memoize, @log, @timeout, @gate
