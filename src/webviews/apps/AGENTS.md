# webviews/apps

Webpack-built webview apps (rebase, settings, welcome).

- **rebase/** – Interactive rebase UI (rebase.html, rebase.ts)
- **settings/** – Settings UI (partials, settings.ts)
- **welcome/** – Welcome page
- **shared/** – Shared: appBase, appWithConfigBase, theme, events, colors, dom, date, snow
- **images/** – Webview images
- **scss/** – Styles
