# webviews/apps/shared

Shared code for webview apps.

- **appBase.ts** – Base app; postMessage bridge
- **appWithConfigBase.ts** – Config-aware base
- **theme.ts** – Theme/vscode API
- **events.ts** – Event handling
- **colors.ts** – Color utilities
- **dom.ts** – DOM helpers
- **date.ts** – Date formatting
- **snow.ts** – Snow effect (welcome)
